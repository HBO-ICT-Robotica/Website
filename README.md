# Website

Installatie:
`npm install`

Kopieer `.env` de bestanden en verwijder `.example`, vul dan de gegevens in

Server starten:
`npm run-script start-dev`

Website builden:
`npm run-script build`

Migraties genereren:
`npm run-script migration:generate`

Migraties toepassen:
`npm run-script migration:apply`
